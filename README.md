# Terraform Labs 

A collection of hands-on DevOps, Cloud, Infrastructure as Code (IaC), Kubernetes, CI/CD, and AWS projects built to gain practical experience in designing, automating, and operating cloud-native platforms.

## About

This repository contains real-world inspired labs and projects covering:

* Infrastructure as Code (Terraform)
* AWS Cloud Services
* Docker & Containerization
* Kubernetes & Amazon EKS
* CI/CD with Jenkins
* GitHub Integration & Automation
* Networking & Security 
* DevOps Best Practices

The goal of this repository is to move beyond theory and build production-style solutions through practical implementation and troubleshooting.

## Featured Projects

### Project 18 - End-to-End CI/CD Pipeline on AWS

Implemented a complete CI/CD platform using Terraform, Jenkins, Docker, Amazon ECR, and Amazon EKS.

**Highlights:**

* Automated infrastructure provisioning using Terraform
* Containerized frontend and backend applications
* Deployed workloads to Amazon EKS
* Configured Amazon ECR for image management
* Built Jenkins pipelines for automated deployments
* Integrated GitHub webhooks for CI/CD automation
* Implemented Kubernetes-based application deployment

**Architecture Flow**

GitHub → Jenkins → Docker Build → Amazon ECR → Amazon EKS → Kubernetes Deployment
